Para rodar esse app 
        Abra a pasta do projeto e em seguida para executar dentro da pasta via terminal este app instale o node e sem seguida execute.

        Instalar as dependencias 
        
        npm install 

        dentro da pasta do projeto rode 

        npm run dev
